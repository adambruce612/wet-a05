Dynamic CSS Masks

Who

This tutorial was created by Michelle Barker.

Where

This tutorial was found on the Codrops site.

When

The turtorial was created on May 4, 2021.

What

This tutorial teaches how to animate CSS masks based on the position of the cursor by implementing GSAP and custom properties.

Why

The concept is interesting because it enables developers to have an eye-catching way to interact with user events.

Demo Download

HTML

The HTML is very simple and straightfoward. To create the effect the content is duplicated. For accessibility the second component is set to: aria-hidden="true" to be screen reader friendly.


        
CSS

        
The CSS for this effect contains a couple of different text effects. First, the text-color and background are
            intentionally set to a high contrast. The drop-shadow filter is also used to create an effect that looks
            like glowing.

        

        
JavaScript

        
The JavaScript for this demo supplies two main things regarding functionality. 1: It is used to set default
            values and, 2, to listen to the mousemove event and updated the mask based on mouse movement.
